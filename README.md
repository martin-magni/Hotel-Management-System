**Hotel Management System**
**Course: BCA - 2nd Semester Project**

**Academic Year: 2023-2024**

**Project Description**
A robust desktop application designed to automate and streamline daily hotel operations. This project was developed as part of the 2nd-semester BCA curriculum to demonstrate proficiency in C# programming, GUI design with Windows Forms, and relational database management.

**Key Features of the project**
Admin Dashboard: Secure login system for authorized staff to manage sensitive hotel data.
Guest Management: Efficiently register, update, and maintain guest profiles.
Room Booking: Real-time tracking of room categories, availability, and reservation status.
Billing System: Automated calculation of stay duration and generation of invoices.
Report Generating: Generate accurate reports using the existing data stored in the files.

**Technologies Used**
Language: C#
Framework: .NET / Windows Forms
IDE: Visual Studio 2022
Database: Local .txt Storing
Version Control: Git & GitHub

**How to Store & Run the project**
Clone the Repository: git clone https://github.com/martin-magni/Hotel-Management-System.git
Open Project: Launch Visual Studio 2022 and open the .sln file.
Create and store all the .txt supporting files.
Update the connection string in the App.config file to match your local server.
Build & Run: Press F5 or click the Start button in Visual Studio.
